# Documentation note

This documentation includes material inherited from upstream `niri`.

In Cidre, the standard compositor binary is `niri-cidre`, and the standard login session is started through `cidre-session`.

When upstream `niri` documentation refers to `niri`, read it as upstream behavior unless the page explicitly mentions Cidre.
